## ✅ 常用 HTTP 状态码整理

| 状态码                | 含义         | 使用场景 / 说明               |
| ------------------ | ---------- | ----------------------- |
| **200 OK**         | 请求成功       | 最常见的返回，数据已正常返回          |
| **201 Created**    | 创建成功       | 成功创建资源（如：POST 新建对象）     |
| **204 No Content** | 请求成功，无返回内容 | DELETE 或 PUT 后无需返回内容时使用 |

---

| 状态码                       | 含义    | 使用场景 / 说明        |
| ------------------------- | ----- | ---------------- |
| **301 Moved Permanently** | 永久重定向 | 页面地址变更，搜索引擎应更新索引 |
| **302 Found**             | 临时重定向 | 页面临时跳转（用户浏览器跳转）  |
| **304 Not Modified**      | 资源未修改 | 浏览器缓存命中，不用重新下载   |

---

| 状态码                        | 含义     | 使用场景 / 说明            |
| -------------------------- | ------ | -------------------- |
| **400 Bad Request**        | 请求错误   | 请求格式错误、参数不合法等        |
| **401 Unauthorized**       | 未认证    | 需要登录（未提供或 token 失效）  |
| **403 Forbidden**          | 被拒绝访问  | 登录了但权限不够，无权访问资源      |
| **404 Not Found**          | 资源不存在  | URL 路径不正确，资源未找到      |
| **405 Method Not Allowed** | 方法不被允许 | 比如接口只支持 POST 却用了 GET |

---

| 状态码                           | 含义      | 使用场景 / 说明        |
| ----------------------------- | ------- | ---------------- |
| **500 Internal Server Error** | 服务器内部错误 | 代码异常、未处理的错误等     |
| **502 Bad Gateway**           | 网关错误    | 通常是代理服务器收到错误响应   |
| **503 Service Unavailable**   | 服务不可用   | 服务宕机、超载或维护中      |
| **504 Gateway Timeout**       | 网关超时    | 上游服务无响应，比如后端处理太慢 |

---

* **2xx → 成功**

  * `200` 常规成功，`201` 是新建，`204` 是没返回内容
* **3xx → 重定向**

  * `301` 永久跳，`302` 临时跳，`304` 用于缓存
* **4xx → 客户端错误**

  * `400` 请求错误，`401` 未登录，`403` 无权限，`404` 不存在
* **5xx → 服务器错误**

  * `500` 程序崩溃，`502/503/504` 常见于网关或微服务问题
