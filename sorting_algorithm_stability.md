> **当两个元素的关键字相等时，排序后它们的相对顺序是否保持不变。**

---

## ✅ 各种常见排序算法稳定性一览表

| 排序算法     | 稳定性   | 说明                             |
| -------- | ----- | ------------------------------ |
| **冒泡排序** | ✅ 稳定  | 相邻元素交换，不改变相等元素顺序               |
| **插入排序** | ✅ 稳定  | 插入时从后往前找位置，相等不交换               |
| **归并排序** | ✅ 稳定  | 合并过程先取左边的相等元素                  |
| **基数排序** | ✅ 稳定  | 按位稳定排序（需使用稳定子排序）               |
| **希尔排序** | ❌ 不稳定 | 插入排序的改进，间隔排序会跳过相邻位置，可能交换相等元素顺序 |
| **选择排序** | ❌ 不稳定 | 选择最小值时会与当前元素交换，可能破坏相等元素顺序      |
| **快速排序** | ❌ 不稳定 | 分区过程中交换可能改变相等元素顺序              |
| **堆排序**  | ❌ 不稳定 | 建堆和调整堆结构时大量交换，不保证相等元素相对顺序      |

- 快速排序
```
(5, a), (3, b), (3, c), (5, d)

```
(4, a), (4, b), (3, c), (2, d)
